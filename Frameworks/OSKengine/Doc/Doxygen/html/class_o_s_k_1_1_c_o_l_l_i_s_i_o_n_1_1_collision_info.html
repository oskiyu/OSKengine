◆ IsColliding()

bool CollisionInfo::IsColliding ( ) const

Devuelve true si hay una colisión entre las entidades, esto es, si existe una colisión entre los colliders de bajo nivel.

Devuelve
True si hay colisión entre las dos entidades.
Comentarios
Si devuelve true, entoncces IsTopLevelColliding también es true.
Si devuelve false, IsTopLevelColliding puede ser true o false.

◆ IsTopLevelColliding()

bool CollisionInfo::IsTopLevelColliding ( ) const

Devuelve true si hay una colisión entre los colliders de alto nivel de las dos entidades. Que haya colisión entre los colliders de alto nivel no implica que haya una colisión entre las entidades, habrá que comprobarlo con IsColliding.

Devuelve
True si hay una colisión entre los colliders de alto nivel de las dos entidades.
Comentarios
Si es true, entonces IsColliding será true o false, dependiendo de cada caso.
Si es false, IsColliding es falso también.
