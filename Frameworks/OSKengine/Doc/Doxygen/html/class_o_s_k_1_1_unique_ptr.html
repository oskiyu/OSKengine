Descripción detallada

template<typename T>
class OSK::UniquePtr< T >

Es dueño de un puntero. El puntero original es eliminado al destruirse el UniquePtr.

Nota
Usarse como miembro deshabilita el constructor de copia.
Precondición
Debe incluirse el tipo que se vaya a usar dentro del UniquePtr.
#include "T.h"
De lo contrario, no se llamará al destructor al eliminarse.
Parámetros del template
TTipo del puntero.
