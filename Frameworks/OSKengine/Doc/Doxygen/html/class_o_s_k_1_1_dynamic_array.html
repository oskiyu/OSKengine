Descripción detallada

template<typename T>
class OSK::DynamicArray< T >

Dynamic array: array que puede cambiar de tamaño.

Nota
Usa memoria contígua.
Atención
No ofrece estabilidad de punteros.
Parámetros del template
TTipo de elementos que van a ser almacenados.
