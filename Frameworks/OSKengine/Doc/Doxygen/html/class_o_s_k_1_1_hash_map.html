Descripción detallada

template<typename TKey, typename TValue, TSize numBuckets = 512>
class OSK::HashMap< TKey, TValue, numBuckets >

Un HashMap representa una colleción que enlaza un valor a otro. Está compuesto internamente por varias colecciones que almacenan los datos.

Atención
No proporciona estabilidad de puntero.
Parámetros del template
TKeyTipo de valor al que se le enlaza otro.
TValueTipo de valor enlazado.
numBucketsNúmero de cubos.
TCollectionTipo de colección de cada cubo.
