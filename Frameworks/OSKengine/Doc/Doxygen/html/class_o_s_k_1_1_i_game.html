Descripción detallada

Esta clase representa la clase principal de un juego de OSKengine. No es estrictamente necesaria para desarrollar con OSKengine, pero es un buen punto de partida. Ofrece una abstracción mínima.

Precondición
Deben sobreescribirse las funciones de CreateWindow() y SetupEngine() de manera obligatoria.

Esta función debe renderizar la imagen final que se representará en la ventana.

Los comandos de renderizado finales deben ejecutarse en el renderpass principal del motor (IRenderer::GetMainRenderpass).

Para construir el frame, se pueden usar las imágenes de los sistemas de renderizado.
