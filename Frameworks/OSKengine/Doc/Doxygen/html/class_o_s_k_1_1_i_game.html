Descripción detallada

Esta clase representa la clase principal de un juego de OSKengine. No es estrictamente necesaria para desarrollar con OSKengine, pero es un buen punto de partida. Ofrece una abstracción mínima.

Precondición
Deben sobreescribirse las funciones de CreateWindow() y SetupEngine() de manera obligatoria.
