Descripción detallada

Clase base para poder importar un modelo GLTF. Se debe crear una clase hija por cada tipo de vértice que se quiera cargar.

Por defecto hay 2:

  • StaticMeshLoader para modelos 3D no animados (Vertex3D).
  • AnimMeshLoader para modelos 3D animados (AnimVertex3D).
