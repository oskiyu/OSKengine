◆ FormatImageDataForGpu()

const TByte * RendererDx12::FormatImageDataForGpu ( const GpuImage *  image,
const TByte *  data,
TSize  numLayers 
)
overrideprotectedvirtual

Debido a razones de alineamiento, es posible que se tengan que desplazar partes de la imagen para que se pueda realizar correctamente una copia de la imagen a la GPU.

Por defecto, devuelve el puntero original.

Parámetros
imageImagen para la que se va a formatear los datos.
dataDatos originales.
Devuelve

Datos con formato.

Si se sobreescribe, devolverá un puntero distinto al original, el cual deberá ser eliminado tras llamar a esta función.
