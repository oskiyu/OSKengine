Descripción detallada

template<typename TComponent>
class OSK::ECS::ComponentContainer< TComponent >

Contenedor para el dynamic array de componentes del tipo dado. Se encarga de manejar la introducción y eliminación de componentes. Los componentes se almacenan en memoria contínua.

Parámetros del template
TComponentTipo del componente.
