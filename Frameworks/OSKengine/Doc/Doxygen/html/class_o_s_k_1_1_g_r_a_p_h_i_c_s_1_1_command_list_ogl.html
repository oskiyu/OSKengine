Registra un buffer intermedio. Como la operación de copia de datos de un buffer intermedio a un buffer final no se realiza hasta que se ejecute la lista de comandos, no podemos eliminar el buffer intermedio hasta la próxima ejecución de la lista de comandos.

Precondición

El buffer debe haber sido creado con GpuSharedMemoryType::GPU_AND_CPU.

void ICommandList::TransitionImageLayout ( GpuImage *  image,
GpuImageLayout  next,
TSize  baseLayer,
TSize  numLayers 
)
inherited

Cambia el layout interno de la imagen en la memoria de la GPU.

Parámetros
nextLayout en el que estará la imagen.
baseLayerPrimera capa a partir de la cual se cambiarán los layouts.
numLayersNúmero de capas a cambiar.
Nota
Se debe cambiar el layout de la imagen antes de ejecutar un comando sobre ella, si su layout actual no coincide con el necesario.
Se considerará el layout anterior el layout que tenía la imagen antes de este comando.
Precondición
El número de capas (numLayers) debe ser mayor que 0.
La lista de comandos debe estar abierta.
