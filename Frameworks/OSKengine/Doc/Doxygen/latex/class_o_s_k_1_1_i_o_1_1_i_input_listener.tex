\doxysubsection{Descripción detallada}
Clase que reacciona a los eventos de entrada. 

Se pueden crear clases derivadas que implementen la funcionalidad deseada para cada caso.

Si las instancias de las clases derivadas se introducen en \mbox{\hyperlink{class_o_s_k_1_1_engine_a5893cf5817c0ab30ae0c0d9761ce71f2}{Engine\+::\+Get\+Input\+Manager()}}, se manejarán de manera automática una vez por frame.
