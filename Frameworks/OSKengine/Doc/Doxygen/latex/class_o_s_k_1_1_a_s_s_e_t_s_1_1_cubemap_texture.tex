\doxysubsection{Descripción detallada}
Representa una textura para un cubemap. Se puede usar para renderizar un skybox. 

\begin{DoxyNote}{Nota}
Representa un array de imágenes con 6 elementos.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
Únicamente para usarse como lectura, no se puede renderizar sobre un cubemap representado por esta clase. 
\end{DoxyWarning}
