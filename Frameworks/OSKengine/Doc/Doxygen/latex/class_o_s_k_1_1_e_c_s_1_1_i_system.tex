\doxysubsection{Descripción detallada}
Un sistema se encarga de ejecutar lógica con los datos de los componentes. Los componentes no son más que colecciones de datos\+: la lógica está en los sistemas. 

\begin{DoxyNote}{Nota}
Cada sistema tiene un Signature, que indica qué componentes debe tener un Game\+Object para ser procesado por el sistema.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
Para que una clase pueda ser usada como sistema, debe tener implementado \mbox{\hyperlink{_i_system_8h_a8f8e44bc705ee82b3070d63d8d702148}{OSK\+\_\+\+SYSTEM(class\+Name)}}. Este debe ser un identificador único\+: no pueden haber dos tipos de sistemas con el mismo class\+Name. 
\end{DoxyWarning}
