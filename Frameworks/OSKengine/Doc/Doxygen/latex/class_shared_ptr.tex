\doxysubsection{Descripción detallada}
\subsubsection*{template$<$typename T$>$\newline
class Shared\+Ptr$<$ T $>$}
Contiene un puntero al que pueden hacer referencia varios \mbox{\hyperlink{class_shared_ptr}{Shared\+Ptr}}. Si el último \mbox{\hyperlink{class_shared_ptr}{Shared\+Ptr}} que hace referencia a un puntero es desstruido, el puntero será eliminado. 

\begin{DoxyNote}{Nota}
La propiedad del puntero está compartida. 
\end{DoxyNote}
