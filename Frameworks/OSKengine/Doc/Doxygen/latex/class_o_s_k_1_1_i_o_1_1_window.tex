\doxysubsection{Descripción detallada}
Clase que representa una ventana. En esta ventana se renderizará el juego. También se encarga del input mediante teclado, ratón, mando, etc... 

\begin{DoxyNote}{Nota}
Esta clase debe usarse de manera estática, accediendo a ella mediante la función Get\+Window(). 
\end{DoxyNote}
