\doxysubsection{Descripción detallada}
Esta clase representa la clase principal de un juego de OSKengine. No es estrictamente necesaria para desarrollar con OSKengine, pero es un buen punto de partida. Ofrece una abstracción mínima. 

\begin{DoxyPrecond}{Precondición}
Deben sobreescribirse las funciones de \mbox{\hyperlink{class_o_s_k_1_1_i_game_af033542cc50642b5e1325ee4c7c93cf7}{Create\+Window()}} y \mbox{\hyperlink{class_o_s_k_1_1_i_game_a3fff81bfbca1fc25efa8a10f2318d46b}{Setup\+Engine()}} de manera obligatoria.
