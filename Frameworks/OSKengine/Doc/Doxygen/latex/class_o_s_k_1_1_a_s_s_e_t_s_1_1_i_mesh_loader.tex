\doxysubsection{Descripción detallada}
Clase base para poder importar un modelo GLTF. Se debe crear una clase hija por cada tipo de vértice que se quiera cargar. 

Por defecto hay 2\+:
\begin{DoxyItemize}
\item \mbox{\hyperlink{class_o_s_k_1_1_a_s_s_e_t_s_1_1_static_mesh_loader}{Static\+Mesh\+Loader}} para modelos 3D no animados (Vertex3D).
\item \mbox{\hyperlink{class_o_s_k_1_1_a_s_s_e_t_s_1_1_anim_mesh_loader}{Anim\+Mesh\+Loader}} para modelos 3D animados (Anim\+Vertex3D). 
\end{DoxyItemize}
