\doxysubsection{Descripción detallada}
\subsubsection*{template$<$typename T$>$\newline
class OSK\+::\+Unique\+Ptr$<$ T $>$}
Es dueño de un puntero. El puntero original es eliminado al destruirse el \mbox{\hyperlink{class_o_s_k_1_1_unique_ptr}{Unique\+Ptr}}. 

\begin{DoxyNote}{Nota}
Usarse como miembro deshabilita el constructor de copia.
\end{DoxyNote}
\begin{DoxyPrecond}{Precondición}
Debe incluirse el tipo que se vaya a usar dentro del \mbox{\hyperlink{class_o_s_k_1_1_unique_ptr}{Unique\+Ptr}}. 
\begin{DoxyCode}{0}
\DoxyCodeLine{\textcolor{preprocessor}{\#include "{}T.h"{}} }

\end{DoxyCode}
 De lo contrario, no se llamará al destructor al eliminarse. 
\end{DoxyPrecond}
