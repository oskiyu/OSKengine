\doxysubsection{Descripción detallada}
Renderiza un mapa de profundidad de la escena desde el punto de vista de una luz direccional. 

Este mapa de profundidad puede usarse para proyectar las sombras de la luz direccional dentro de una escena.

Para usar un \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_shadow_map}{Shadow\+Map}}, en el bucle de renderizado\+:
\begin{DoxyItemize}
\item Iniciar un renderpass con el render target Shadow\+Map\+::\+Get\+Shadows\+Render\+Target.
\item Enlazar el material \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_shadow_map_ac65819749ffb51ef5ce805ac35b7910d}{Shadow\+Map\+::\+Get\+Shadows\+Material}}.
\item Enlazar la instancia de material \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_shadow_map_ad7263c5e330f9148ba87ebff3aae13b3}{Shadow\+Map\+::\+Get\+Shadows\+Material\+Instance}}.
\item Renderizar todos los objetos que deban proyectar sombras.
\item Finalizar el renderpass. 
\end{DoxyItemize}
