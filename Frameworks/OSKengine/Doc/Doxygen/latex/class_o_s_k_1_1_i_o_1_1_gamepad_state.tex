\doxysubsection{Descripción detallada}
Representa el estado actual de un gamepad conectado a la plataforma. 

\begin{DoxyNote}{Nota}
Puede no estar conectado, se debe comprobar que esté conectado.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
En caso de no estar conectado, los valores devueltos pueden ser incoherentes. Comprobar siempre que el gamepad está conectado. 
\end{DoxyWarning}
