\doxysubsection{Descripción detallada}
Representa un sprite\+: una textura que puede ser renderizada. También es un componente que permite a un Game\+Object tener un sprite asociado. 

\begin{DoxyNote}{Nota}
Se encarga de manejar el material 2D por defecto.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
Antes de ser usado para renderizado, se deben establecer la instancia del material, la cámara y la imagen. 
\end{DoxyWarning}
