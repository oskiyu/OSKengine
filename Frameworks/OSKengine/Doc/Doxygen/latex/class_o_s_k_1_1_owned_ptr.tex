\subsubsection*{template$<$typename T$>$\newline
class OSK\+::\+Owned\+Ptr$<$ T $>$}
Es dueño de un puntero. 

\begin{DoxyNote}{Nota}
El puntero original NO es eliminado al destruirse el \mbox{\hyperlink{class_o_s_k_1_1_owned_ptr}{Owned\+Ptr}}. 
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
Se desaconseja su uso como miembro de clase. No copia el recurso si se copia la clase, lo que puede generar problemas. 
\end{DoxyWarning}
