\doxysubsection{Descripción detallada}
Clase que simplifica el proceso de renderizado de imágenes 2D sin tener que usar el sistema Render\+System2D, lo cual te da un mayor control sobre el orden en el que se renderizan los sprites. 

\begin{DoxyNote}{Nota}
Antes de usar los comandos de renderizado, se debe llamar a \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_sprite_renderer_a103d94e314984318d0d99e7d07752601}{Sprite\+Renderer\+::\+Set\+Command\+List}} y \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_sprite_renderer_a2ae2a44a9c2fb615b298afb0d8fbf603}{Sprite\+Renderer\+::\+Begin}}. 

Despues de finalizar el renderizado, se debe llamar a \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_sprite_renderer_a614b8a5f4e11f2b4a32cfd0710781c9f}{Sprite\+Renderer\+::\+End}}.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
No se deben ejecutar otros comandos de renderizado desde que se llama a \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_sprite_renderer_a2ae2a44a9c2fb615b298afb0d8fbf603}{Sprite\+Renderer\+::\+Begin}} hasta que se llama a \mbox{\hyperlink{class_o_s_k_1_1_g_r_a_p_h_i_c_s_1_1_sprite_renderer_a614b8a5f4e11f2b4a32cfd0710781c9f}{Sprite\+Renderer\+::\+End}}. 
\end{DoxyWarning}
