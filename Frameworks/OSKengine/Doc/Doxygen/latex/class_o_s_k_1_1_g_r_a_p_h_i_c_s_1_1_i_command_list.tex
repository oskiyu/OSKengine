\doxysubsection{Descripción detallada}
Una lista de comandos contiene una serie de comandos que serán enviados a la GPU para su ejecución. 

La lista es creada por una pool de comandos, y se introduce en una cola de comandos para su ejecución. 

\begin{DoxyNote}{Nota}
Los comandos serán ejecutados en orden una vez se envíe la lista a la GPU. 
\end{DoxyNote}
