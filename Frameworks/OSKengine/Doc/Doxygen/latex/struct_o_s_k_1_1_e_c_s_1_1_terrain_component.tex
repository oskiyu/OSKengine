\doxysubsection{Descripción detallada}
Componente que se puede usar para renderizar un terreno. El terreno debe haber sido antes generado. 

A la hora de generarse, se generará un terreno con dimensiones \{ 1, 0, 1 \}\+: los vértices no tendrán altura ninguna. Las coordenadas de texturas se asignarán para que una textura esté entera dentro del terreno. 

\begin{DoxyNote}{Nota}
Si se le quiere dar relieve al terreno, se debe hacer a través de shaders.
\end{DoxyNote}
\begin{DoxyWarning}{Atención}
Se debe establecer la instancia del material (\mbox{\hyperlink{struct_o_s_k_1_1_e_c_s_1_1_terrain_component_acdee489fde34e4aa92c379db15e1c3e5}{Terrain\+Component\+::\+Set\+Material\+Instance()}}). 
\end{DoxyWarning}
