Descripción detallada

template<typename TEvent>
requires IsEcsEvent<TEvent>
class OSK::ECS::ConsumerSystem< TEvent >

Un sistema consumidor es un sistema que se encarga de responder a eventos. Estos sistemas se ejecutan después de los sistemas productores y antes que los sistemas puros.

Parámetros de plantilla
TEventTipo de evento al que está suscrito.
