Define qu dispositivo tiene acceso a la memoria.

Puede definir una memoria exclusiva para la GPU (mayor rendimiento) o una memoria accesible tanto desde la CPU como la GPU.

Valores de enumeraciones
GPU_AND_CPU 

Compartido. til para recursos que van a ser sobreescritos a menudo, o recursos de un solo uso a los que vamos a escribir informacin directamente, como buffers intermedios.

GPU_ONLY 

Exclusivo para la GPU. Mejor rendimiento.

Atencin
No se puede escribir directamente desde la CPU: se debe crear un buffer intermedio al que tengamos acceso (GpuSharedMemoryType::GPU_AND_CPU), y despus copiar sus contenidos usando la cola de comandos.
