◆ DxcCreateInstance()

DXC_API_IMPORT HRESULT __stdcall DxcCreateInstance ( _In_ REFCLSID rclsid,
_In_ REFIID riid,
_Out_ LPVOID * ppv )

Creates a single uninitialized object of the class associated with a specified CLSID.

Parámetros
rclsidThe CLSID associated with the data and code that will be used to create the object.
riidA reference to the identifier of the interface to be used to communicate with the object.
ppvAddress of pointer variable that receives the interface pointer requested in riid. Upon successful return, *ppv contains the requested interface pointer. Upon failure, *ppv contains NULL.

While this function is similar to CoCreateInstance, there is no COM involvement.
