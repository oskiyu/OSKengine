◆ GetElapsedTime()

TDeltaTime StopWatch::GetElapsedTime ( ) const
Precondición
Debe haberse llamado al menos una vez a Start().
Devuelve
Tiempo (en segundos) medidos por el StopWatch:
  • Si el estado actual es State::RUNNING, entonces devuelve el tiempo transcurrido desde la última llamada a Start().
  • Si el estado actual es State::STOPPED, entonces devuelve el tiempo transcurrido entre la última llamada a Start() y la última llamada a Stop().
