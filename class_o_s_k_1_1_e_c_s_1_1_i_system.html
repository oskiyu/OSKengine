Descripción detallada

Un sistema se encarga de ejecutar lógica.

Atención
Para que una clase pueda ser usada como sistema, debe tener implementado OSK_SYSTEM(className). Este debe ser un identificador único: no pueden haber dos tipos de sistemas con el mismo className.
