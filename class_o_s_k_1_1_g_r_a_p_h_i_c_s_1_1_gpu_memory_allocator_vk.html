◆ CreateStagingBuffer()

OwnedPtr< GpuBuffer > IGpuMemoryAllocator::CreateStagingBuffer ( USize64 size,
GpuBufferUsage usage = GpuBufferUsage::UPLOAD_ONLY )
inherited

Crea un buffer temporal con el tamaño dado.

Este buffer alojará memoria en una región donde la CPU tiene acceso, de tal manera que se podrán enviar datos a esta región, y después la GPU podrá copiar esta región de memoria a una región de memoria de video más rápida.

Parámetros
sizeTamaño del buffer, en bytes.
usageUso que se le va a dar al buffer, además de como staging buffer.
Devuelve
Staging buffer.
Nota
El buffer se colocará en un bloque con GpuSharedMemoryType::GPU_AND_CPU.
Postcondición
El buffer siempre tendrá al menos los usos GpuBufferUsage::TRANSFER_SOURCE | GpuBufferUsage::UPLOAD_ONLY.
