Descripción detallada

Representa una región de memoria dedicada a un único recurso.

Puede escribirse información sobre la GPU a través del subbloque. Para poder escribir, primero es necesario mapear la memoria necesaria, y desmapearla al terminar de escribir.

Hay dos modos de escritura:

  • Write: escribe la información, y hace avanzar un cursor interno, de tal manera que la siguiente vez que se llame a esta función, se escribirá con un offset, para no sobreescribir la información antes escrita.
  • WriteOffset: ignora el cursor interno.
