Descripción detallada

Una lista de comandos contiene una serie de comandos que serán enviados a la GPU para su ejecución. La lista es creada por una pool de comandos, y se introduce en una cola de comandos para su ejecución.

Nota
Los comandos serán ejecutados una vez se envíe la lista a la GPU.

Copia la imagen guardada en un buffer a una imagen en la GPU. Para poder subir una imagen a la GPU, primero debemos subir la imagen a un buffer intermedio, y después copiamos el contenido de ese buffer intermedio a la imagen final de la GPU.

Parámetros
sourceBuffer con los contenidos de la textura.
destImagen final.
layerCapa a la que se copiará los contenidos. Por defecto: 0.
offsetOffset, en bytes, respecto al inicio del buffer.
Precondición
El buffer de origen debe haber sido creado con GpuBufferUsage::TRANSFER_SOURCE.
La imagen de destino debe haber sido creada con GpuImageUsage::TRANSFER_DESTINATION.
Si la imagen de destino tiene niveles de mip-map, entonces debe haber sido creada con GpuImageUsage::TRANSFER_SOURCE.
La imagen de destino debe tener el layout GpuImageLayout::TRANSFER_DESTINATION.
La lista de comandos debe estar abierta.
Postcondición
El layout de la imagen después de efectuarse la copia segirá siendo GpuImageLayout::TRANSFER_DESTINATION.
Si la imagen de destino tiene niveles de mip-map, estos estarán correctamente generados.
Nota
Si el número de niveles de mip de dest es 1, el tipo de comando es de TRANSFERENCIA.
Precondición
Si el número de niveles de mip de dest es 1, el pool de comandos a partir del cual se ha creado la lista debe soportar, al menos, CommandsSupport::TRANSFER.
Si el número de niveles de mip de dest es 1, la cola de comandos sobre la que se envía la lista debe soportar, al menos, CommandsSupport::TRANSFER.
Nota
Si el número de niveles de mip de dest es mayor que 1, el tipo de comando es de TRANSFERENCIA y GRÁFICOS.
Precondición
Si el número de niveles de mip de dest es mayor que 1, el pool de comandos a partir del cual se ha creado la lista debe soportar, al menos, CommandsSupport::TRANSFER y CommandsSupport::GRAPHICS.
Si el número de niveles de mip de dest es mayor que 1, la cola de comandos sobre la que se envía la lista debe soportar, al menos, CommandsSupport::TRANSFER y CommandsSupport::GRAPHICS.

Implementado en OSK::GRAPHICS::CommandListDx12 y OSK::GRAPHICS::CommandListVk.
