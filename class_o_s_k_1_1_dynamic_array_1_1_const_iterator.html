Descripción detallada

template<typename T>
class OSK::DynamicArray< T >::ConstIterator

Clase que representa un elemento de un DynamicArray.

Precondición
La colección generadora debe tener estabilidad de puntero mientras el iterador siga vivo.
Atención
Si se modifica el dynamic array original, queda en un estado inválido.
Si se modifica la localización en memoria del dynamic array original, queda en un estado inválido.
