Descripción detallada

Clase que reacciona a los eventos de entrada.

Se pueden crear clases derivadas que implementen la funcionalidad deseada para cada caso.

Si las instancias de las clases derivadas se introducen en Engine::GetInputManager(), se manejarán de manera automática una vez por frame.

◆ SetMinAxisValue()

void IInputListener::SetMinAxisValue ( float value)

Establece el valor mínimo, en valor absoluto, que debe tener el eje para que se considere activo.

Se usa para evitar problemas de drifting.

Nota
Debe estar entre 0 y 1.
Si se introduce un valor fuera del rango válido, se hará clamp a dicho valor.
