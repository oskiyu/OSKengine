const IGpuImageView * GpuImage::GetView ( const GpuImageViewConfig & viewConfig) const
inherited

Obtiene un image view con las características dadas.

Parámetros
channelCanal de la imagen accedido por el view.
arrayTypeConfigura si es una única imagen o si el view afecta a todo un array.
baseArrayLevelCapa base representada.
layerCountNúmero de capas representadas (si arrayType es SampledArrayType::ARRAY).
baseMipLevelMip level más detallado.
topMipLevelMip level menos detallado.
usageUso del view.
Devuelve
View con las características dadas.
Precondición
Si layerCount > 1, entonces arrayType debe ser SampledArrayType::ARRAY.
topMipLevel >= baseMipLevel.
Los mip levels indicados por el rango topMipLevel y baseMipLevel deben existir en la imagen original.
Excepciones
ImageViewCreationExceptionSi hay algún error al crear el view.
