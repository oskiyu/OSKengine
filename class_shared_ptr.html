Descripción detallada

template<typename T>
class SharedPtr< T >

Contiene un puntero al que pueden hacer referencia varios SharedPtr. Si el último SharedPtr que hace referencia a un puntero es desstruido, el puntero será eliminado.

Nota
La propiedad del puntero está compartida.
Parámetros de plantilla
TTipo del puntero almacenado.
