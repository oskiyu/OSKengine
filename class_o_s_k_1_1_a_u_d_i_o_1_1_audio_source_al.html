◆ SetBuffer()

void AudioSourceAl::SetBuffer ( const Buffer & buffer)

Enlaza el source a un buffer. Necesario para poder reproducir audio. El audio reproducido será el almacenado en el buffer.

Parámetros
bufferBuffer con el audio.
Precondición
buffer debe haber sido correctamente inicializado.
buffer debe haber sido correctamente asignado con los datos del audio.
Excepciones
AudioExceptionSi no se inicializó el Source con Init.
AudioExceptionSi ocurre algún error al establecer el buffer.
