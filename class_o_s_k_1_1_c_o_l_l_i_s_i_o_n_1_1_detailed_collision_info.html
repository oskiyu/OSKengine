◆ GetContactPoints()

const DynamicArray< Vector3f > & DetailedCollisionInfo::GetContactPoints ( ) const

Devuelve todos los puntos de contacto entre ambos colliders. El número de puntos dependerá de cada caso.

Devuelve
Lista con los puntos de contacto (en espacio del mundo).
Precondición
Debe haber una colisión entre ambos colliders (IsColliding() debe ser true).
Nota
Si IsColliding() es true, entonces habrá al menos un punto de contacto.
Si IsColliding() es false, entonces devuelve una lista vacía.
